# 🚗 帮你选车 - 使用说明

## 快速开始

1. **打开首页**  
   双击 `index.html` 或在浏览器中打开

2. **查看 Token**  
   顶部显示当前可用次数（首次赠送 3 次）

3. **开始测试**  
   点击"开始选车"按钮

4. **回答问题**（5 题）
   - 你的购车预算是多少？
   - 主要用车场景是？
   - 能源类型偏好？
   - 最看重车辆的哪些方面？
   - 品牌偏好？

5. **查看结果**  
   系统会根据你的答案，自动过滤并推荐匹配的车型！

---

## 匹配逻辑

每辆车会显示**匹配度百分比**：

- ✅ **预算匹配** (40%) - 价格在你的预算范围内
- ✅ **场景匹配** (20%) - SUV→家庭，轿车→通勤等
- ✅ **能源匹配** (20%) - 燃油/纯电/增程符合你的选择
- ✅ **品牌匹配** (20%) - 国产/合资/进口符合你的偏好

## 外网访问（部署指南）

### 方法一：Vercel（推荐 ⭐⭐⭐⭐⭐）

1. 注册 https://vercel.com
2. 导入 `/workspace/car-selector` 文件夹
3. 一键部署，自动生成 HTTPS 链接
4. 分享给你朋友使用！

### 方法二：Netlify

1. 注册 https://netlify.com
2. 拖拽整个文件夹到后台
3. 自动生成 HTTPS 链接

### 方法三：GitHub Pages

1. 创建 GitHub 仓库
2. 上传 `car-selector` 文件夹
3. Settings → Pages → 开启
4. 获取 `username.github.io/car-selector` 链接
